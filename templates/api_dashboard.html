Integration Guide

Use your API key to make requests to the Globle-1 Unified Endpoint.

Endpoint

POST /api/v1/chat/completions

Python Example (Using Library)

from globlexgptai import GlobleXGPTAiClient

# Initialize the client with your key
client = GlobleXGPTAiClient(api_key="your_globle_api_key")

# Generate a response
response = client.generate("Hello Globle-1, how are you?")
print(response)

cURL Example

curl -X POST {{ url_for('index', _external=True) }}api/v1/chat/completions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "globle-1",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'

Follow these steps to start using Globle-1 in your projects.

1. Install Python (if not installed)
